# RoboJailBench

RoboJailBench is a benchmark for evaluating adversarial attacks and defenses in embodied robotic agents.

- Project website: https://purseclab.github.io/benchmark-for-robotics-security/
- GitHub repository: https://github.com/purseclab/benchmark-for-robotics-security
- Croissant metadata: https://purseclab.github.io/benchmark-for-robotics-security/croissant.json

## Authors

Doguhan Yeke, Yanming Zhou, Leo Y. Lin, Hongyu Cai, Antonio Bianchi, and Z. Berkay Celik. All authors are affiliated with Purdue University.
